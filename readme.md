# E-Book Recommender System
โครงงานคอมพิวเตอร์ ปริญญาวิทยาศาสตร์บัณฑิต สาขาวิทยาการคอมพิวเตอร์ ภาควิชาวิทยาการคอมพิวเตอร์ มหาวิทยาลัยขอนแก่น

ระบบแนะนำหนังสืออิเล็กทรอนิกส์ สำหรับนักศึกษา ภาควิชาวิทยาการคอมพิวเตอร์ มหาวิทยาลัยขอนแก่น (หลักสูตร 2560) โดยใช้ขั้นตอนวิธีการแบบลูกผสม (Hybrid) ที่นำขั้นตอนวิธีการอิงตามเนื้อหา (Content-based) เพื่อช่วยคัดกรองหนังสืออิเล็กทรอนิกส์ตามเนื้อหาของหนังสือที่ผู้ใช้สนใจ และวิธีการกรองร่วมกัน (Collaborative filtering) ซึ่งใช้ความชอบของผู้ใช้จากการให้คะแนนความพึงพอใจต่อหนังสือ อีกทั้งยังใช้พฤติกรรมการอ่าน คำที่ใช้ค้นหาหนังสืออิเล็กทรอนิกส์ ในการคัดกรองหนังสืออิเล็กทรอนิกส์ที่มีลักษณะหรือคุณสมบัติที่ใกล้เคียงกับหนังสืออิเล็กทรอนิกส์ที่ผู้ใช้สนใจ มาผสมผสานกันเพื่อแนะนำหนังสืออิเล็กทรอนิกส์ให้กับผู้ใช้โดยอัตโนมัติ
